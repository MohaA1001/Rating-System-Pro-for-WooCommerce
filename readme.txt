Replace WooCommerce reviews with a stars system featuring professional-style breakdowns, manual star counts, and Top Rated badges.

== Description ==

**Rating System Pro** is a powerful WooCommerce enhancement that transforms the standard product reviews into a professional, stars-only rating system. Inspired by market-leading and high-converting e-commerce sites, it provides a clean, visual representation of customer feedback.

Whether you want to display manual "seed" ratings for new products or offer a streamlined experience for real customers, this plugin gives you total control over the rating display and submission process.

= Key Features =

*   **Stars-Only Mode** – Remove the wordy review textarea; customers submit only their star rating for a faster experience.
*   **Manual Rating Distribution** – Full control from the product edit screen. Manually enter the count of 5★, 4★, 3★, 2★, and 1★ ratings for any product.
*   **Visual Rating Breakdown** – Dynamic, animated progress bars showing the distribution of ratings on the product page.
*   **Top Rated Badge** – Automatically display a customizable badge (e.g., "Top Rated" or "Best Seller") when a product's average rating exceeds your set threshold (e.g., 4.5+).
*   **Guest Rating Support** – Allow visitors to rate products without logging in. Duplicate votes are prevented using session tracking.
*   **AJAX-Powered Submission** – Ratings are submitted instantly. The average score and progress bars update in real-time without a page refresh.
*   **Highly Customizable Colors** – Match your brand exactly by picking custom colors for stars, progress bars, and badges directly from the settings.
*   **Per-Product Overrides** – Choose to use global ratings or specific manual distributions for individual products.
*   **Developer Friendly** – Clean OOP structure, HPOS compatible, and includes hooks for further customization.

== Installation ==

1. Search for "Rating System Pro for WooCommerce" on the add Plugins screen.
2. Click install & activate.
3. Go to **WooCommerce → Rating System** to configure your global settings and colors.
4. (Optional) Edit any product to enter manual star counts under the **Manual Rating Distribution** meta box.

== Frequently Asked Questions ==

= Does this replace the standard WooCommerce Reviews? =
Yes. It removes the default reviews tab and replaces it with a dedicated "Ratings" tab that features the breakdown widget and the new rating form.

= Is it compatible with High-Performance Order Storage (HPOS)? =
Absolutely. The plugin is built with modern WooCommerce standards and declares full compatibility with HPOS and custom order tables.

= Can I use it alongside normal reviews? =
By default, it is designed to replace them for a cleaner UI.
